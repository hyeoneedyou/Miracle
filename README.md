## Commit Message Rule

| type     | content                            |
| -------- | ---------------------------------- |
| feat     | 새로운 기능 추가                   |
| fix      | 버그 수정                          |
| docs     | 문서 관련                          |
| refactor | 코드 리팩토링                      |
| test     | 테스트 코드                        |
| add      | 기능, 문서 외의 코드 추가          |
| correct  | 문법 오류, 타입 변경, 오타 등 수정 |
| modify   | 수정                               |
